# **Codeback**

기존 Discord나 Zoom과 같은 서비스에서 화면 공유를 통해 스터디를 진행하면서 겪는
불편함(화면을 벗어난 코드 확인, 테스트 케이스 입력)을 개선하기위해 실시간 공동 편집,
컴파일, 화상회의 기능을 한 번에 제공하는 서비스
